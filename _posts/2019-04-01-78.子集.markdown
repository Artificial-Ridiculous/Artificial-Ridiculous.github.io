给定一组不含重复元素的整数数组 nums，返回该数组所有可能的子集（幂集）。

示例：  

### 输入: 
```python  
nums = [1,2,3]  
```
### 输出:  
```python
[  
  [3],  
  [1],  
  [2],  
  [1,2,3],  
  [1,3],  
  [2,3],  
  [1,2],  
  []  
]  
```

### 解答：  

```python
def subsets(self, nums: List[int]) -> List[List[int]]:
    res = [[]]
    for num in nums:
        res += [i+[num] for i in res]
    return res
```
