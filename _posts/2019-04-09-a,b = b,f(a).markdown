当写形如`a,b = c,f(a)`时，注意，`b=f(a)`用的`a`依然是`a=c`之前的那个`a`，类似C语言的`a++`，而不是`++a`。  
举例：

```python
>>> a = 3
>>> a,b = 100,a+1
>>> b
4
>>> a
100
```

```python
>>> a = 3
>>> a = 100
>>> b = a+1
>>> a
100
>>> b
101
```
