连接操作及支持的条件运算符：

连接操作 | where |  on  
-|-|-
,|o|x

---

连接操作效果示例：

原表：

`a`：

```
+------+--------+
|   id | name   |
|------+--------|
|    1 | alice  |
|    2 | bob    |
|    3 | cissy  |
|    5 | edard  |
+------+--------+
```

`b`：

```
+------+--------+
|   id | name   |
|------+--------|
|    1 | alice  |
|    2 | bob    |
|    4 | david  |
|    5 | edard  |
|    9 | iris   |
+------+--------+
```

---

逗号内连接

```sql
select * from a ,b where a.id = b.id
```

```
+------+--------+------+--------+
|   id | name   |   id | name   |
|------+--------+------+--------|
|    1 | alice  |    1 | alice  |
|    2 | bob    |    2 | bob    |
|    5 | edard  |    5 | edard  |
+------+--------+------+--------+
```

---

join内连接

```sql
select * from a join b where a.id = b.id
```

```
+------+--------+------+--------+
|   id | name   |   id | name   |
|------+--------+------+--------|
|    1 | alice  |    1 | alice  |
|    2 | bob    |    2 | bob    |
|    5 | edard  |    5 | edard  |
+------+--------+------+--------+
```

---

left join 左连接

```sql
select * from a left join b on a.id = b.id
```

```
+------+--------+--------+--------+
|   id | name   |     id | name   |
|------+--------+--------+--------|
|    1 | alice  |      1 | alice  |
|    2 | bob    |      2 | bob    |
|    3 | cissy  | <null> | <null> |
|    5 | edard  |      5 | edard  |
+------+--------+--------+--------+
```

---

right join 右连接

```sql
select * from a right join b on a.id = b.id
```

```
+--------+--------+------+--------+
|     id | name   |   id | name   |
|--------+--------+------+--------|
|      1 | alice  |    1 | alice  |
|      2 | bob    |    2 | bob    |
|      5 | edard  |    5 | edard  |
| <null> | <null> |    4 | david  |
| <null> | <null> |    9 | iris   |
+--------+--------+------+--------+
```
