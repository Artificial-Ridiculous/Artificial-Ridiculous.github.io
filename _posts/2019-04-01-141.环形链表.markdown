# 

给定一个链表，判断链表中是否有环。

示例：  

### 输入1:

### 输出1:

```python
True
```

### 输入2:   
![ListNode]({{ "/PNG/141_circularlinkedlist_test3.png" | absolute_url }})

### 输出2:  
```python
False
```

### 解答：  

```python
def detectCycle(self, head):
    if head is None : return None
    if head.next is None : return None

    slow = fast = head
    steps = 0
    flag = False
    while(fast and fast.next):
        fast = fast.next.next
        slow = slow.next
        steps += 1
        if (fast == slow):
            flag = True
            break
    if not flag:
        return None
    else:
        while(head!=slow):
            slow = slow.next
            head = head.next
        return head
```
